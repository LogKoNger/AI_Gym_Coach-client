# AI Gym Coach - Personalized Fitness & Nutrition Assistant

An intelligent chatbot powered by AI that provides personalized workout plans, nutrition guidance, and fitness assessments for gym enthusiasts. The system analyzes user inputs such as weight, height, and lifestyle choices through interactive questionnaires to deliver tailored recommendations that match individual fitness goals and physical conditions.
